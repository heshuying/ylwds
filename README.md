## 码值
用户类型： 01:个人客户;02:商户；03平台；04：供应商
用户状态：0-待审核 1-正常 2-锁定；3-审核不通过；4-禁用
订单状态：1-待支付    2-待发货   3-待收货   4-待评价   5-售后中    6-待结算    7-结算中     8-已结算

结算状态： 1-待确认；2-待平台更改；3-待开票   4-待付款   5-已完结
